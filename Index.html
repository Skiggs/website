Hotlike Miami

  • Github Link

A 2D Top Down Shooter game which heavily focuses on Enemy AI. I have made an attempt to build this project from ground up. It is made on C++ using stl and sfml for rendering shapes and textures, the rest is made from scratch. The project involves the integration of complex enemy AI. A simple Data-Oriented Design approach has been followed in this project for handling different events and messages by the enemies.

This layer involves the development of complex behavior AI. The behavior includes different enemy states like Patrolling, go for check, consume food, take cover from enemy, shoot, throw grenades, dodge grenades, revive friend. The aim is to make the enemy think like a human and decide his actions through analysis of the surroundings. For example, if the enemy is bare handed then it will look for a weapon around - if the weapon is in danger zone then it will decide to stay behind the cover and wait until the weapon is not in danger zone or just go to the enemy for a fist fight. If an enemy dies then the closest one to the died enemy goes for reviving and the rest keeps fighting with the player. The enemy is even reactive to the surroundings - consuming poisonous sandwich and then running all the way to the toilet. All these states are handled by state machine which is being attached on the enemies. The AI is very much inspired from the game Hitman.
